# PPO_14 Design: Lower GAE Lambda + More Envs

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Break the EV ceiling (~0.55) that has plateaued PPO_13 since 400k steps by reducing advantage estimate variance and increasing rollout diversity.

**Architecture:** Three targeted hyperparameter changes to `python/train.py` ‚Äî `gae_lambda` reduction (0.95‚Üí0.90) to reduce variance in advantage estimates, `n_envs` default increase (8‚Üí16) for more diverse rollout data per update, `eval_episodes` increase (50‚Üí100) for cleaner model selection. LR slightly reduced (5e-5‚Üí5e-6 to 3e-5‚Üí3e-6) since PPO_13's clip fraction consistently ran at 0.17, just at the upper edge of target. All other architecture unchanged (features_dim=128, pattern_board channel, dense reward).

**Tech Stack:** Python, stable-baselines3 PPO, gymnasium. All changes in `python/train.py` and `docs/2026-02-24-rl-agent-design.md`.

## Background

PPO_13 plateau diagnosis at 1M steps:
- Eval peaked at **16.09** (new best vs PPO_11's 14.58) but stalled since 400k
- EV ceiling **0.47‚Äì0.57** ‚Äî critic can't model value well enough to guide further improvement
- Rollout reward **FLAT** ~7.3‚Äì7.4 for 300k+ steps
- Policy gradient loss **positive** ‚Äî policy no longer finding improving directions
- Clip fraction **0.17** ‚Äî consistently at upper edge of 0.1‚Äì0.2 target

Root cause: GAE with Œª=0.95 produces high-variance advantage estimates (effectively near-Monte-Carlo returns). The critic can't fit noisy long-horizon returns well enough to push EV past 0.55. More envs per rollout also gives the critic more diverse training signal.

| Param | PPO_13 | PPO_14 | Rationale |
|-------|--------|--------|-----------|
| `gae_lambda` | 0.95 | **0.90** | Reduce advantage variance ‚Üí help critic ‚Üí higher EV |
| `n_envs` default | 8 | **16** | 2√ó rollout diversity per update |
| `eval_episodes` default | 50 | **100** | Reduce eval noise (swing was ¬±3 points) |
| LR | 5e-5‚Üí5e-6 | **3e-5‚Üí3e-6** | clip_frac was 0.17, slightly lower start |
| Everything else | ‚Äî | unchanged | pattern_board, dense reward, features_dim=128, n_steps=2048, gamma=0.99, target_kl=0.01 |
